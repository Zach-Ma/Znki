# Znki

## Description

> (server side of)flashcard app based on Anki.

informed dependencies:

- `nestjs`
- `typeorm`
- `prisma`
- `@nestjs/swagger`,`@nestjsx/crud`

## Installation

```bash
npx prisma init
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
